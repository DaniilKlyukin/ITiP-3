# Информационные технологии и программирование (3 семестр)

Учебный репозиторий для студентов направления **«Прикладная математика и информатика»**. В этом семестре основное внимание уделяется расширенным возможностям языка C#, разработке графических интерфейсов (Windows Forms), работе с данными и внешними библиотеками.

**Преподаватель:** Клюкин Даниил Анатольевич  
**Кафедра:** ПМиИТ  
**GitHub:** [DaniilKlyukin](https://github.com/DaniilKlyukin)

## 📚 Содержание лекций

| № | Тема | Ключевые концепции |
|:---:|:---|:---|
| 1 | **[Делегаты и события](Лекции/Лекция%201.pdf)** | `delegate`, анонимные методы, лямбда-выражения, `Action`, `Func`, `Predicate`, события (`event`). |
| 2 | **[Программные интерфейсы](Лекции/Лекция%202.pdf)** | Понятие API (Web API, REST, SOAP), GUI, различие между библиотекой и фреймворком. |
| 3 | **[Windows Forms (WinForms)](Лекции/Лекция%203.pdf)** | Жизненный цикл формы, основные компоненты (Button, TextBox, и др.), привязки `Anchor` и `Dock`. |
| 4 | **[Работа с данными и метаданными](Лекции/Лекция%204.pdf)** | `DateTime`, сериализация JSON (Newtonsoft), атрибуты (`Serializable`, `DataMember`), компонент `PropertyGrid`. |
| 5 | **[Рефлексия и плагины](Лекции/Лекция%205.pdf)** | Исследование типов в рантайме, `Assembly`, позднее связывание, динамическая загрузка плагинов через интерфейсы. |
| 6 | **[Регулярные выражения](Лекции/Лекция%206.pdf)** | Класс `Regex`, кванторы, группы захвата, валидация текста (email, телефоны), замена по шаблону. |
| 7 | **[Графика и вычисления](Лекции/Лекция%207.pdf)** | Рисование (`Graphics`, `Pen`, `Brush`), анимация (Timer), графики `ScottPlot`, библиотека `MathNet.Numerics`. |

## 🧪 Лабораторные работы (ЛР)

*   **ЛР №1:** Использование делегатов и событий в логике приложения.
*   **ЛР №2:** Разработка многооконного интерфейса на Windows Forms.
*   **ЛР №3:** Сериализация объектов и работа с метаданными через `PropertyGrid`.
*   **ЛР №4:** Динамическое расширение функционала (система плагинов через рефлексию).

---

## 📝 Практические работы (ПР)

*   **ПР №1:** Валидация пользовательского ввода с помощью Regex.
*   **ПР №2:** Построение математических графиков функций и гистограмм.
*   **ПР №3:** Решение систем линейных уравнений (СЛАУ) средствами MathNet.
*   **ПР №4:** Создание простых анимаций и работа с растровой графикой (`Bitmap`).

---

## 🛠 Инструменты и требования
*   **Среда разработки:** Visual Studio 2022 (Community).
*   **Платформа:** .NET 8.0 - 10.0
*   **Библиотеки (NuGet):**
    *   `Newtonsoft.Json` — для работы с форматом JSON.
    *   `ScottPlot.WinForms` — для построения интерактивных графиков.
    *   `MathNet.Numerics` — для научных и инженерных вычислений.
