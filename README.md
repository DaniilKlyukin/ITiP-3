# Курс "Информационные технологии и программирование 3"

Данный репозиторий содержит учебные материалы для студентов 3 семестра направления "Прикладная математика". Курс посвящен изучению продвинутых возможностей языка C#, включая делегаты, события, Windows Forms, сериализацию, рефлексию, построение графиков и работу с 2D-графикой.

## 📚 Содержание курса

### Лекционные материалы
Лекции охватывают ключевые темы для создания современных desktop-приложений.
- [Лекция 1](Лекции/ИТиП%203%20лек%201.pptx) - Делегаты, анонимные методы, лямбда-выражения, события.
- [Лекция 2](Лекции/ИТиП%203%20лек%202.pptx) - Программный интерфейс (API), графический интерфейс (GUI), разница между API, Framework и Library.
- [Лекция 3](Лекции/ИТиП%203%20лек%203.pptx) - Windows Forms приложения: основные компоненты, привязки `Anchor` и `Dock`, события.
- [Лекция 4](Лекции/ИТиП%203%20лек%204.pptx) - Работа с датами и временем (`DateTime`), сериализация в JSON, атрибуты, элемент `PropertyGrid`.
- [Лекция 5](Лекции/ИТиП%203%20лек%205.pptx) - Рефлексия (Reflection), исследование типов, позднее связывание, создание плагинов.
- [Лекция 6](Лекции/ИТиП%203%20лек%206.pptx) - Регулярные выражения (`Regex`), классы символов, квантификаторы, группы захвата.
- [Лекция 7](Лекции/ИТиП%203%20лек%207.pptx) - Рисование на форме (GDI+), создание анимации, построение графиков (`ScottPlot`), математические вычисления (`MathNet.Numerics`).

### Лабораторные работы
Комплексные задания, требующие применения знаний из нескольких тем.

| № | Тема | Задание |
|---|------|---------|
| 1 | [События](Лабораторные/Лабораторная%20работа%201.docx) | Разработка библиотеки классов с событиями для заданной предметной области (библиотека, ресторан, финансы и др.). |
| 2 | [Разработка Windows Forms приложений](Лабораторные/Лабораторная%20работа%202.docx) | Создание графического интерфейса для системы управления (товары, покупатели, заказы) с использованием библиотеки классов. |
| 3 | [Построение графиков](Лабораторные/Лабораторная%20работа%203.docx) | Разработка приложения для построения 2D-графиков функций с помощью ScottPlot. Реализация маркеров, отображения значений, настройки внешнего вида. |
| 4 | [Работа с 2D графикой](Лабораторные/Лабораторная%20работа%204.docx) | Построение алгебраического фрактала "Бассейн Ньютона" с интерактивными возможностями (приближение, настройка параметров, сохранение). |

### Практические работы
Небольшие упражнения для отработки конкретных навыков.

| № | Тема | Задание |
|---|------|---------|
| 1 | [События](Практики/Практическая%20работа%201.docx) | Консольные приложения на события: "Светофор", "Умный дом", "Кнопка с историей". |
| 2 | [Сериализация данных](Практики/Практическая%20работа%202.docx) | Разработка Windows Forms приложения для регистрации пользователей с валидацией данных через регулярные выражения и сохранением в JSON с помощью рефлексии. |

### Экзаменационные материалы
- [Билеты к экзамену](Билеты.docx) - 12 экзаменационных билетов, включающих теоретические вопросы и практические задания по разработке Windows Forms приложений, визуализации и сериализации.

## 🎯 Цели курса

- Изучение событийно-ориентированного программирования на примере делегатов и событий.
- Освоение разработки desktop-приложений с графическим интерфейсом (Windows Forms).
- Получение навыков работы с форматами данных (JSON) и их сериализацией/десериализацией.
- Знакомство с рефлексией и созданием расширяемых приложений (плагины).
- Применение регулярных выражений для валидации и обработки текста.
- Развитие навыков визуализации данных с помощью графиков и анимации.

## 📋 Требования к выполнению работ

1.  Работы выполняются в среде Visual Studio 2026.
2.  Код должен быть аккуратно оформлен, содержать XML-комментарии к классам и методам.
3.  При разработке Windows Forms приложений необходимо следовать принципам разделения логики (интерфейс, логика, данные).
4.  Отчеты по лабораторным работам оформляются в соответствии с методическими указаниями.
5.  Работы сдаются в установленные сроки.
