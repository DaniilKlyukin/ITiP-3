# Информационные технологии и программирование (3 семестр)

Учебный репозиторий для студентов направления **«Прикладная математика и информатика»**. В этом семестре основное внимание уделяется расширенным возможностям языка C#, разработке графических интерфейсов (Windows Forms), рефлексии, регулярным выражениям и работе с научными библиотеками.

**Преподаватель:** Клюкин Даниил Анатольевич  
**Кафедра:** ПМиИТ  
**GitHub:** [DaniilKlyukin](https://github.com/DaniilKlyukin)

## 🛠 Инструменты и требования
*   **Среда разработки:** Visual Studio 2022.
*   **Платформа:** .NET 8.0 - 10.0.
*   **Используемые библиотеки (NuGet):**
    *   `Newtonsoft.Json` (Сериализация данных).
    *   `ScottPlot.WinForms` (Интерактивные графики).
    *   `MathNet.Numerics` (Линейная алгебра и интерполяция).

## 🎓 Аттестация (3 семестр)
*   **Форма контроля:** Зачет с оценкой.
*   **Условия допуска:** Отсутствие задолженностей по лабораторным и практическим работам.
*   **Критерии:** Уверенное владение механизмами делегирования, понимание принципов работы WinForms и умение использовать рефлексию для динамических задач.
